# lib_gdal

Crystal wrapper for GDAL. Only a subset of the library is supported for now.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     lib_gdal:
       github: crmorello/lib_gdal
   ```

2. Run `shards install`

## Usage

```crystal
require "lib_gdal"
```

You will need GDAL installed.
